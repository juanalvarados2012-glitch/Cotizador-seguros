## 6. La reunión (guion exprés, con énfasis en escala)

1. **Apertura + agenda.**
2. **Descubrimiento** → volumen de plantillas (¡mucho!), cómo unifican criterio
   entre suscriptores (y más con la fusión), cuellos de botella.
3. **Demo en vivo** → subir → autollenar → IA → editar/aprender → exportar.
4. **Valor** → enfatiza **consistencia de criterio a escala** + métricas
   (% autollenado, ahorro de tiempo). A una empresa grande le importa medir.
5. **Oferta** → piloto acotado a un equipo (30 días, criterio de éxito); luego
   **Pro $499/mes** o **Enterprise desde $899/mes** (ilimitado + soporte
   prioritario). Onboarding $350.
6. **Objeciones** → seguridad de datos, integración con sus sistemas (aclara: tu
   app trabaja sobre el Excel del broker, no reemplaza nada), tiempos.
7. **Cierre** → define **equipo piloto + métricas + fecha**.
8. **Gracias** + correo de seguimiento.

> 🎓 A una empresa grande no le vendes "ahorro de horas" a secas: le vendes
> **escala + consistencia + datos para medir**. Habla su idioma.
